# Portfolio Brief — Jonathan Vera

## Objetivo del proyecto

Crear un portfolio profesional moderno, minimalista y enfocado en software empresarial.

El objetivo NO es parecer diseñador gráfico ni frontend creativo.

El objetivo es transmitir:
- experiencia real construyendo software empresarial,
- capacidad técnica sólida,
- pensamiento orientado a negocio,
- experiencia backend/full stack,
- automatización de procesos,
- integraciones complejas,
- manejo de datos y sistemas administrativos.

El portfolio debe sentirse:
- profesional,
- limpio,
- moderno,
- rápido,
- minimalista,
- enfocado en ingeniería de software.

La sensación general debe ser:
> “Este desarrollador sabe construir soluciones reales para empresas.”

## Framework
Usar Astro.

## Estilos
Usar Tailwind CSS.
